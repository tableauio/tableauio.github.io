Scaffolding

Provide scaffolding code of configuration Loader and Checker.

Metadata Protoconf

Create a powerful IDL(Interface Description Language) Protoconf to define the structure of Excel/XML, based on Protobuf(proto3).

Cross-platform

The tableau toolset can be run on Windows, macOS, Linux, and so on. Which is powered by Go.

Multiple formats

Convert Excel/XML to multiple formats: JSON(protojson), TEXT(prototext), and BIN(protowire).

Multilingual

Support C++, C#, Go, Java, Python, Typescript/Javascript, and so on. Which is powered by Protobuf.
