Tableau

Tableau是一款功能强大的现代化配置转换器。基于Protobuf (proto3)，它能够将 Excel/CSV/XML 转换成多种格式：JSON，Text 和 Bin。

Features

⚡ 快

数秒内可转换上千个 Excel/CSV/XML 文件。

🛠️ 脚手架

提供 protoc 插件，生成 Loader 和 Checker 脚手架代码。

📄 元数据

基于 Protobuf 扩展，精心设计了 IDL Protoconf，以便于定义和描述 Excel/CSV/XML 文件结构.

🧊 跨平台

Tableau相关工具链可运行于多种操作系统：Windows, macOS 和 Linux.

📚 多格式

可以将 Excel/CSV/XML 导出多种文件格式：JSON, Text 和 Bin.

📝 多语言

支持 C++, C#, Go, Java, Python, Lua, TS/JS 等多种语言接入，这得益于 Protobuf 的强大生态。
