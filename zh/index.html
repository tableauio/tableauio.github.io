Tableau配置转换器

Tableau是一款利用Protobuf(proto3)的强大特性开发的现代化配置转换器，能够将Excel/XML转换成多种格式：JSON(protojson)，TEXT(prototext)和BIN(protowire)。

开始

Open-source MIT Licensed. Tableau v0.9.4

快

可以在数秒内转换数千个Excel/XML文件到指定输出格式，这是利用Go语言开发的益处。

Scaffolding

Provide scaffolding code of configuration Loader and Checker. Powered by protoc plugin system.

Metadata Protoconf

The powerful IDL Protoconf (based on Protobuf) is designed to define the structure of Excel/CSV/XML.

Cross-platform

The tableau toolset can be run on Windows, macOS, Linux, and so on.

Multiple formats

Convert Excel/CSV/XML to multiple formats: JSON, Text, and Wire.

Multilingual

C++, C#, Go, Java, Python, TS/JS, and so on, are supported, due to Protobuf.
